翻看了 chatGPT 的记录，开始学习 Python是 4 月 7 号，到今天为止，差不多三周。在此期间，认真地读了 Python 教材一本（《Python 教程：从入门到实践》，就教材质量而言出类拔萃，即使横跨多领域也是超五星基本的教材），理解了 Python 的基本概念和语法结构、面向对象的概念和使用（类和实例）、Django 框架的基本使用（模型、路由、视图函数和模板）。并提纲挈领地理解了 React 和 NextJS 的概念和哲学。

目前的初步成果是这个博客，使用 nextjs 技术，基于 [Tailwind Nextjs Starter Blog](https://github.com/timlrx/tailwind-nextjs-starter-blog) 主题进行了本地化的修改。三周时间，也算是填坑了之前一直想选用较为流行的技术框架搭建个人博客的小心愿。

还记得当初迷失在“文件即路由”、“客户端/服务器渲染”等让人云里雾里的术语森林里面的情形，在凑凑合合的理念之下，也就让基于 Beautiful Jekyll 模板的 GitHub Pages 跌跌撞撞地用了这许多年，徒留下 gmail 收件箱中每隔一段时间就会发送的因技术过时而引起的风险提醒邮件。开心的是，此刻的我，脑海中已经有了一张框架图：代码和文字构成的信息流，是如何从此刻手边的文字编辑器，通过 GitHub 仓库中转到 nextjs 服务器，再辗转呈现在读者屏幕中浏览器的页面。喜欢这一路走来的流畅感，也喜欢这种懂了概念之后一览众山小的感觉。

回到题目，上述过程无比琐碎但其对我的意义在于，迄今为止的人生存档，主线任务中关于我这个人的“画像”，是与上述这些码农日常毫不相干的。经济学专业与财务专家的职业路径，与上述的任何一个词汇都毫无相关，如果对日常生活中的任何一个朋友或业务伙伴说起这些技术术语，我们将只会陷入仿佛在说不同语言的巴别塔困境。

但游戏既然在继续，也就是可以新开存档的。无关勇气与坚持，我觉得自己只是找到了一种即使失败了也不会怎么样的念头，以及就在手边随时可以辅导我的 AI 老师而已。

行舟知水，这是 Deepseek 帮我取的博客名字，寓意是保持行进，保持对水（即周围所有）的感知。这两年如果说有什么对人生帮助重大的感悟，大概就是“意义在于行动之中”，“做一件事最好的时间是现在”。

最后，控制住了使用 AI 润色这篇文章的念头，下午刚使用 chatGPT 快速写了两篇在 medium 发表的英文文章。读了之后，只是默默地想保留自己手工打造的自留地，就当种点菜给自己吃吧。

附注：本博客新建于此日，2025年4月25日。之前的几篇文章搬运自原托管于 GitHub Pages 的老博客。
